# 使用手册

本文档详细介绍 RofenClaw 的各项功能和使用方法。

## 🏠 主页

启动应用后，首先进入主页。主页显示：

- **系统状态概览** - Gateway 连接状态、分身数量等
- **快捷操作** - 快速创建对话、管理分身
- **最近活动** - 最近的对话和操作记录

## 💬 对话功能

### 发起对话

1. 点击侧边栏的「对话」进入对话页面
2. 在底部输入框输入消息
3. 按 `Enter` 或点击发送按钮
4. 等待 AI 分身回复

### 对话管理

- **新建对话** - 点击左上角「+」按钮
- **切换对话** - 点击左侧对话列表中的项目
- **删除对话** - 右键点击对话，选择删除
- **重命名对话** - 双击对话标题进行编辑

### 对话操作

- **复制回复** - 点击消息右侧的复制按钮
- **重新生成** - 点击重新生成按钮获取新回复
- **保存到记忆** - 将重要对话保存到长期记忆

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `Ctrl + N` | 新建对话 |
| `Ctrl + S` | 保存对话 |

## 🎭 分身管理

### 什么是分身？

分身是具有特定角色和能力配置的 AI 助手。每个分身可以：
- 使用不同的 AI 模型
- 拥有独立的记忆和上下文
- 配置特定的技能和工具

### 创建分身

1. 进入「分身管理」页面
2. 点击「新建分身」按钮
3. 填写分身信息：
   - **名称** - 分身的显示名称
   - **描述** - 分身的角色描述
   - **模型** - 使用的 AI 模型
   - **系统提示** - 分身的行为指令
4. 点击「保存」

### 分身配置

| 配置项 | 说明 |
|--------|------|
| 模型 | 选择 AI 模型（如 GPT-4, Claude 等） |
| 温度 | 控制回复的创造性（0-1） |
| 最大令牌 | 回复的最大长度 |
| 系统提示 | 定义分身的角色和行为 |

### 切换分身

在对话页面，点击顶部分身名称，选择要使用的分身。

## 🛠️ 技能库

### 什么是技能？

技能是扩展 AI 能力的插件，可以为分身添加特定功能，如：
- 网页搜索
- 代码执行
- 文件操作
- API 调用

### 安装技能

1. 进入「技能库」页面
2. 切换到「市场」标签
3. 浏览或搜索技能
4. 点击「安装」

### 管理技能

- **启用/禁用** - 切换技能开关
- **配置** - 点击技能进入配置页面
- **卸载** - 点击卸载按钮移除技能

### 内置技能

| 技能 | 功能 |
|------|------|
| 网页搜索 | 搜索互联网信息 |
| 代码执行 | 运行代码片段 |
| 文件操作 | 读写文件系统 |
| 日志记录 | 记录操作日志 |

## 📊 状态监控

### 系统状态

状态监控页面显示：

- **Gateway 状态** - 连接状态、延迟
- **资源使用** - CPU、内存使用情况
- **会话统计** - 对话数量、令牌使用
- **活动日志** - 最近的系统活动

### 3D 分身视图

页面中央显示动态的 3D 分身形象：
- **空闲状态** - 平静呼吸动画
- **思考状态** - 活跃的发光效果
- **工作状态** - 快速脉冲动画

## 💻 开发者工具

### 代码编辑器

开发者页面提供代码编辑和执行环境：

1. 选择编程语言
2. 在编辑器中输入代码
3. 点击「运行」执行代码
4. 查看输出结果

### 支持的语言

- JavaScript
- Python
- Bash
- PowerShell

### 历史记录

- 查看执行历史
- 快速加载历史代码
- 清除历史记录

## 📝 日志查看

### 日志级别

| 级别 | 颜色 | 说明 |
|------|------|------|
| DEBUG | 绿色 | 调试信息 |
| INFO | 蓝色 | 一般信息 |
| WARN | 黄色 | 警告信息 |
| ERROR | 红色 | 错误信息 |

### 日志操作

- **刷新** - 获取最新日志
- **过滤** - 按级别或关键词过滤
- **搜索** - 搜索日志内容
- **导出** - 导出日志文件
- **清空** - 清除所有日志

### 自动刷新

开启自动刷新后，日志每 5 秒自动更新。

## ⚙️ 系统设置

### 常规设置

| 设置项 | 说明 |
|--------|------|
| 语言 | 界面语言 |
| 开机启动 | 是否随系统启动 |
| 显示启动画面 | 是否显示启动画面 |
| 最小化到托盘 | 关闭窗口时最小化到系统托盘 |

### Gateway 设置

| 设置项 | 说明 |
|--------|------|
| 端口 | Gateway 监听端口 |
| 认证模式 | 认证方式 |
| Token | 访问令牌 |

### 模型设置

| 设置项 | 说明 |
|--------|------|
| 默认模型 | 新对话使用的模型 |
| 最大并发 | 最大并发请求数 |
| 子分身并发 | 子分身最大并发数 |

### API 密钥管理

在设置页面可以管理各 AI 提供商的 API 密钥：
- OpenAI
- Anthropic
- Moonshot
- 其他兼容提供商

## 🔔 通知

### 系统通知

应用会在以下情况发送通知：
- Gateway 连接状态变化
- 长时间任务完成
- 错误发生时

### 通知设置

在设置页面可以配置：
- 启用/禁用通知
- 通知声音
- 通知位置

## ⌨️ 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + C` | 快速打开对话 |
| `Ctrl + Shift + S` | 打开设置 |
| `Ctrl + Q` | 退出应用 |

## 📱 系统托盘

应用最小化后会在系统托盘显示图标：

- **单击** - 显示/隐藏窗口
- **右键** - 显示快捷菜单
  - 打开主窗口
  - 新建对话
  - 退出应用

## 🔒 安全建议

1. **保护 API 密钥** - 不要分享配置文件
2. **使用 Token 认证** - 生产环境启用认证
3. **定期更新** - 保持应用和依赖最新
4. **备份配置** - 定期备份重要配置
