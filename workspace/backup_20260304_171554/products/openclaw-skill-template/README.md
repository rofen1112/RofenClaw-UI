# OpenClaw 技能开发模板

一个完整的 OpenClaw Agent Skill 模板，帮助你快速创建标准化的技能文件。

## 包含内容

1. **SKILL.md** - 标准化的技能描述文件
2. **示例脚本** - 可执行的示例代码
3. **目录结构** - 推荐的文件组织方式
4. **工具函数** - 常用工具封装

## 适用场景

- 快速创建新的 Agent Skill
- 学习 SKILL.md 规范
- 统一团队技能文件格式

## 使用方法

1. 复制本模板到 `~/.openclaw/workspace/skills/你的技能名/`
2. 修改 SKILL.md 中的元数据和描述
3. 添加你的脚本到 `scripts/` 目录
4. 在 TOOLS.md 中记录配置信息

## 目录结构

```
my-skill/
├── SKILL.md          # 技能描述（必需）
├── scripts/          # 可执行脚本
├── assets/           # 图片、数据文件等
└── README.md         # 额外说明
```
