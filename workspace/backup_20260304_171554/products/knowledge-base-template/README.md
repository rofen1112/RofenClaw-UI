# 个人知识库模板 (PARA 方法)

一套基于 PARA 方法的个人知识管理系统模板，适用于 Obsidian、Logseq 等工具。

## 什么是 PARA

PARA 是由 Tiago Forte 提出的知识组织方法：

- **P**rojects (项目) - 有明确目标和截止日期的任务
- **A**reas (领域) - 需要长期维护的责任范围
- **R**esources (资源) - 感兴趣的主题和参考材料
- **A**rchives (归档) - 已完成或不活跃的项目/领域

## 使用方法

1. **收集** → 想法先进入 Inbox
2. **整理** → 定期将 Inbox 内容分类
3. **执行** → 在 Projects 中推进任务
4. **维护** → 在 Areas 中持续改进
5. **参考** → 在 Resources 中查找资料
6. **归档** → 完成的项目移到 Archives

## 配套规则

- 每个项目必须有明确的完成标准
- 每周回顾 Areas 的状态
- 每月清理 Inbox
- 每年归档已完成项目
