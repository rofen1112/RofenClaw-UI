# AI 对话优化提示词包

一套经过实战验证的提示词模板，让 AI 输出更结构化、更精准、更实用。

## 包含内容

### 1. 结构化输出模板

让 AI 按固定格式返回结果，便于后续处理：

```markdown
## 角色
你是专业的[领域]专家。

## 任务
[描述具体任务]

## 输出格式
请严格按照以下 JSON 格式返回：
{
  "summary": "一句话总结",
  "key_points": ["要点1", "要点2", "要点3"],
  "action_items": ["行动1", "行动2"],
  "confidence": 0.95
}

## 输入内容
[用户输入]
```

### 2. 代码生成优化模板

让 AI 生成可直接运行的生产级代码：

```markdown
请编写 [语言] 代码实现以下功能：
[功能描述]

要求：
1. 包含完整错误处理
2. 添加类型注解
3. 编写 docstring 文档
4. 提供使用示例
5. 考虑边界情况
6. 代码风格遵循 [PEP8/Google Style]

输出格式：
- 先给出整体思路
- 然后提供完整代码
- 最后说明注意事项
```

### 3. 分析类任务模板

用于数据分析、文档解析等场景：

```markdown
分析以下 [内容类型]：
```
[内容]
```

请提供：
1. 核心结论（3点以内）
2. 关键数据/证据
3. 潜在风险/注意事项
4. 建议行动

用 Markdown 格式输出，关键信息加粗。
```

### 4. 创意生成模板

用于头脑风暴、创意发散：

```markdown
主题：[主题]

请生成 10 个 [创意/方案/标题]，要求：
- 多样性：覆盖不同角度
- 创新性：不拘泥于常规
- 可行性：有落地可能

格式：
| 序号 | 创意 | 亮点 | 适用场景 |
|-----|-----|-----|---------|
```

### 5. 决策辅助模板

帮助分析选项、做出决策：

```markdown
决策场景：[描述]

选项：
A. [选项A描述]
B. [选项B描述]
C. [选项C描述]

请从以下维度分析：
1. 成本效益
2. 风险评估
3. 长期影响
4. 执行难度

输出：
- 对比表格
- 推荐方案及理由
- 实施建议
```

### 6. 翻译优化模板

专业级翻译提示词：

```markdown
将以下内容翻译成 [目标语言]：
```
[原文]
```

要求：
1. 保持专业术语准确
2. 符合目标语言习惯
3. 保留原文格式
4. [如有]使用 [正式/口语/技术] 风格

输出格式：
翻译：[译文]
术语对照：[关键术语表]
备注：[翻译说明]
```

## 使用技巧

1. **替换占位符** - 把 [方括号] 内容替换为你的实际需求
2. **灵活组合** - 多个模板可以组合使用
3. **持续优化** - 根据结果调整提示词
4. **保存复用** - 把常用的保存为个人模板
